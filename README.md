# lrinsurances-demo
LIFE Insurances 7.3 Demo stack

## Getting Started

### Requirements

- Docker 19+

### Run

```
docker-compose up -d
```

### Explore

Play with Liferay at http://localhost:8080 and deploy your applications under `./liferay/deploy`.
